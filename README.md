# Astronomical-datascience-with-python

This repository contains the tutorial materials from the 14-Day Astronomical Data Science with Python Workshop conducted by Spartificial.
The notebooks and examples introduce the fundamentals of working with astronomical datasets using Python, covering data loading, visualization, preprocessing, and basic analysis techniques commonly used in astronomy.

## Dataset

The workshop uses datasets stored in the FITS (Flexible Image Transport System) format, which is the standard file format used in astronomy for storing scientific data.

A FITS file typically consists of one or more Header Data Units (HDUs). Each HDU contains:

Header – Stores metadata describing the observation, telescope, instrument settings, dimensions, coordinate information, and other relevant details.
Data – Contains the actual scientific data, such as images, spectra, or tables.

Multiple HDUs can exist within a single FITS file, allowing different types of related astronomical data to be stored together.
